# Quick Start: Editing Chapter Content

## How to Edit a Chapter in 3 Steps

### Step 1: Open the Story Viewer
1. Generate or load a story with completed chapters
2. Click **"View Story"** button
3. Select the chapter you want to edit from the sidebar

### Step 2: Enter Edit Mode
1. Scroll to the bottom of the chapter
2. Click the **"Edit Chapter Content"** button (blue button with pencil icon)
3. A large text editor will appear with your chapter content

### Step 3: Make Changes and Save
1. Edit the text in the textarea
2. Click **"Save Changes"** when done
3. Your chapter will update and return to read mode

## Quick Tips

✅ **Auto-Save**: If you switch chapters while editing, your changes are automatically saved

✅ **Undo Available**: Use "Undo Last Revision" to restore previous versions

✅ **Cancel Anytime**: Click "Cancel" to discard changes without saving

✅ **Paragraph Format**: Keep each paragraph on its own line for proper formatting

## Common Use Cases

```

3. Click "Save Changes"
```

### Add More Description
```

```


## Buttons Explained

| Button | What It Does |
|--------|-------------|
| **Edit Chapter Content** | Opens the editor for the current chapter |
| **Save Changes** | Saves your edits and returns to read mode |
| **Cancel** | Discards changes and returns to read mode |
| **Undo Last Revision** | Restores the previous version of the chapter |

## When to Use Editing vs. Regeneration

### Use **Editing** When:
- Making small fixes (typos, grammar)
- Adding minor details
- Adjusting specific sentences or paragraphs
- You know exactly what you want to change

### Use **Regeneration** When:
- The entire chapter needs major changes
- You want the AI to rewrite with new instructions
- The tone or style is completely wrong
- You want to try a different approach to the scene

## Safety Features

🔒 **Revision History**: Every edit creates a backup of the previous version

🔒 **Auto-Save**: Changes are saved when switching chapters

🔒 **Undo Function**: Restore previous versions anytime

🔒 **Cancel Option**: Discard changes before saving

## Keyboard Shortcuts

While editing:
- **Tab**: Insert tab (in textarea)
- **Ctrl/Cmd + A**: Select all text
- **Ctrl/Cmd + Z**: Undo (within textarea)
- **Ctrl/Cmd + Y**: Redo (within textarea)

## Troubleshooting

**Q: The "Edit Chapter Content" button is missing**
- Make sure you're in the Story Viewer (not the chapter generation screen)
- Verify the chapter has been generated and has content
- Check that you're viewing a completed chapter

**Q: My changes disappeared**
- Check if you clicked "Cancel" instead of "Save Changes"
- Try using "Undo Last Revision" to see if the changes are in history
- Make sure you clicked "Save Changes" before closing the browser

**Q: The "Save Changes" button is grayed out**
- This means no changes have been made to the content
- Make at least one change to enable the button

**Q: Can I edit multiple chapters at once?**
- No, you can only edit one chapter at a time
- However, switching chapters auto-saves your current edits

## Example Workflow

### Scenario: Improving a Generated Chapter

1. **Read the chapter** in the Story Viewer
2. **Identify areas** that need improvement
3. **Click "Edit Chapter Content"**
4. **Make targeted edits**:
   - Fix any awkward phrasing
   - Add more vivid descriptions
   - Enhance dialogue
   - Adjust pacing
5. **Click "Save Changes"**
6. **Read the updated chapter** to verify improvements
7. **If needed**, click "Undo Last Revision" to try again

### Scenario: Collaborative Writing

1. **Generate the chapter** with AI
2. **Edit to add your personal touch**:
   - Add inside jokes or references
   - Include specific details from your world
   - Adjust character voices to match your vision
3. **Save and continue** to the next chapter
4. **Repeat** for a blend of AI efficiency and human creativity
